# 💳 Duxlr - Advanced Currency Exchange Simulation

**Duxlr** is a high-fidelity fintech dashboard simulation designed to demonstrate a seamless currency exchange experience between **USD** and **DZD**. Built with a focus on modern UI/UX and professional financial aesthetics.

## 🚀 Features
* **Real-time Exchange Logic:** Instant conversion based on simulated live market rates.
* **Virtual Card Issuance:** Sleek, animated card display for a premium feel.
* **Dynamic Ledger:** Live transaction history tracking.
* **Responsive Design:** Fully optimized for mobile and desktop using Tailwind CSS.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS (JIT Engine).
* **Icons & Fonts:** FontAwesome 6.0, Inter/Roboto Fonts.
* **Logic:** Pure Vanilla JavaScript.
